# AI Expense Analyzer

A full-stack web application to track and analyze personal expenses with intelligent insights.

## Features
- Add and delete expenses
- Category-wise spending analysis
- Interactive pie chart visualization
- AI-based insights on spending behavior

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS
- Chart.js

## How to Run
pip install flask pandas  
python app.py  
http://127.0.0.1:5000
